## 基本类型:按值操作(直接操作的是值),所以也叫做值类型
## 引用类型:操作的是堆内存的地址(按引用地址操作的)

```
let n = [10,20];
let m = n;
let x = m;
m[0] = 100;
x = [30,40];
x[0] = 200;
m = x;
m[1] = 300;
n[2] = 400;
console.log(n); =>n=100,20,400
console.log(m); =>m=200,300
console.log(x); =>x=200,300
```
